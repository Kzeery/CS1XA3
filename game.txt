ls = look at surroundings
cd Location = move to a new location
cd .. = backtrack
less ITEM = interact with things
pwd = see where you are
cd ~ = go home
mv OBJECT NEWLOCTAION = move things
help = list of available spells
rm = removem
mkdir = make directory
grep Phrase Item = search for phrase in an item
touch Item = create item
cp Item NewItem = copy an item with new name NewItem
cp Item ../ NewItem = copy item to previous dir with new name
sudo = superuser do (perform an action with superuser function [requires password])
add Locker = add a new locker to home
